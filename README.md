## 💼 Core Projects & Achievements

### 1. ニュース発信ネイティブスマホアプリ開発（2025.03 - 2025.09）
基本設計からアジャイルで開発したネイティブニュースアプリです。
* **位置情報による機能**: ユーザーの現在位置から**半径3キロ以内**で発生した事件やニュースをマップ上にリアルタイム表示する機能を実装。
* **データベースパフォーマンス改善**: MySQLで地理情報データを高速処理するため、`explain analyze`を用いてスロークエリを徹底解析。適切なインデックス設計を行い、レスポンス速度を大幅に向上させました。
* **マーケティング＆収益化**: Google Analytics等の分析ツールやAdMobによる広告組み込みを実装。

### 2. 行政向け申請Webシステム開発（2025.01 - 2025.03）
申請プロセスのシステム改修と新機能の追加を担当。
* **ログAPIの実装**: ユーザー動線を正確に追跡可能なログAPIをバックエンド（Java）に構築し、フロントエンドへの統合を実施。
* **データ設計の最適化**: どのステータスをどの型でDBに保持すべきかチームと議論・設計を行い、後の機能拡張が容易な堅牢な設計を実現。

### 3. コールセンター向けQAポータル構築＆社内DX（2025.10 - 2025.12）
ノーコード・ローコードツールを活用した社内業務の自動化。
* **問い合わせ連携の自動化**: SharePointとPowerAutomateを連携し、QAが回答されたら即座に担当者へMicrosoft Teamsで自動通知されるシステムを構築。運用業務のスピードアップとDX化に貢献。
